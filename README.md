Visit my live site at: [jayyidsptr.github.io](https://jayyidsptr.github.io)

## ✨ Key Features
- **Premium UI/UX**: Clean, modern, and high-contrast design with glassmorphism effects.
- **Multilingual Support**: Fully localized in **Indonesian** and **English** using a custom i18n system.
- **Interactive Background**: Dynamic 3D particle background powered by **Three.js**.
- **Responsive Layout**: Optimized for all devices, from mobile phones to large desktops.
- **Micro-Animations**: Smooth scroll reveal effects and interactive cursor follow.
- **Project Showcase**: Detailed cards with live demo links and hover effects.
- **Mentoring Section**: Integrated call-to-action for 1-on-1 coding mentorship via WhatsApp.

## 🛠️ Technology Stack
- **Core**: HTML5, Vanilla JavaScript (ES6+).
- **Styling**: Vanilla CSS with custom design tokens.
- **Icons**: [Lucide Icons](https://lucide.dev/).
- **Animations**: GSAP (optional integration) & Native CSS Animations.
- **3D Graphics**: [Three.js](https://threejs.org/).
- **Localization**: Custom lightweight JavaScript i18n implementation.
